fopen的第二个参数常见的形式有：
“rt”　　　　　　只读打开一个文本文件，只允许读数据 
“wt”　　　　　　只写打开或建立一个文本文件，只允许写数据
“at”　　　　　　追加打开一个文本文件，并在文件末尾写数据
“rb”　　　　　　只读打开一个二进制文件，只允许读数据
“wb”　　　　 　 只写打开或建立一个二进制文件，只允许写数据
“ab” 　　　　 　追加打开一个二进制文件，并在文件末尾写数据
“rt+”　　　　　 读写打开一个文本文件，允许读和写
“wt+”　　　　　 读写打开或建立一个文本文件，允许读写
“at+”　　　　　 读写打开一个文本文件，允许读，或在文件末追加数据
“rb+”　　　　　 读写打开一个二进制文件，允许读和写 
“wb+”　　　　　 读写打开或建立一个二进制文件，允许读和写
“ab+” 　　　　　读写打开一个二进制文件，允许读，或在文件末追加数据

得到文件位置指针当前位置相对于文件首的偏移字节数：
ftell(fp);//在随机方式存取文件时，由于文件位置频繁的前后移动，程序不容易确定文件的当前位置。该函数对大于231-1文件，即：2.1G以上的文件操作时可能出错。

文件位置查找：
int fseek(FILE *stream, long offset, int fromwhere);//函数设置文件指针stream的位置。如果执行成功，stream将指向以fromwhere（偏移起始位置：文件头0(SEEK_SET)，当前位置1(SEEK_CUR)，文件尾2(SEEK_END)）为基准，偏移offset（指针偏移量）个字节的位置。如果执行失败(比如offset超过文件自身大小)，则不改变stream指向的位置。
fseek函数和lseek函数类似，但lseek返回的是一个off_t数值，而fseek返回的是一个整型。返回值：成功，返回0，失败返回-1，并设置errno的值，可以用perror()函数输出错误。

整型转换为数字组成的字符串：
头文件：stdio.h
itoa(num, str, 10);//函数有3个参数：第一个参数是要转换的数字，第二个参数是要写入转换结果的目标字符串，第三个参数是转移数字时所用的基数。在上例中，转换基数为10。

清空结构体或数组
memset(需要清空的指针,被填入的字符,被填入空间的大小);//eg：memset(a,20,sizeof(a));作用是在一段内存块中填充某个给定的值，它是对较大的结构体或数组进行清零操作的一种最快方法。

从文件中读入链表或数组
链表：fread(链表指针, 单个链表大小,1,文件指针eg：fp);//完成后移动链表指针到下一个
数组：fread（数组名，整个数组大小，数组个数eg：a[100],则这个空填100,文件指针);

链表或数组写入文件
链表：fwrite(链表指针,单个链表大小,1,文件指针eg：p);//完成后移动链表指针到下一个   
数组：fwrite(数组名,数组大小,数组个数eg：a[100],则这个空填100,文件指针);

获取时间信息
time_t nowtime;
struct tm *timeinfo;
time(&nowtime);//获取系统时间
timeinfo=localtime(&nowtime);//改变时间格式
timeinfo->tm_sec; /* 秒 C 取值区间为[0,59] */
timeinfo->tm_min; /* 分 - 取值区间为[0,59] */
timeinfo->tm_hour; /* 时 - 取值区间为[0,23] */
timeinfo->tm_mday; /* 一个月中的日期 - 取值区间为[1,31] */
timeinfo->tm_mon+1; /* 月份 - 取值区间为[1,12] */
timeinfo->tm_year+1900; /* 年份*/
timeinfo->tm_wday; /* 星期 C 取值区间为[0,6]，其中0代表星期天，1代表星期一，以此类推 */
timeinfo->tm_yday; /* 从每年的1月1日开始的天数 C 取值区间为[0,365]，其中0代表1月1日，1代表1月2日，以此类推 */
timeinfo->tm_isdst; /* 夏令时标识符，实行夏令时的时候，tm_isdst为正。不实行夏令时的进候，tm_isdst为0；不了解情况时，tm_isdst()为负。*/
